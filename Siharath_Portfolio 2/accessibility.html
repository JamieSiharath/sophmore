Web Accessibility Checklist
User characteristics Accessible design tip
Unable to see Code all images with ALT text
Unable to perceive colo Avoid using color alone to convey information
Unable to use mouse Be sure all website features can be acessed using keyboard
Unable to hear Add captions to multimedia
Prone to having seizures Avoid content that flashes
Easily distractible Keep the design simple
